=============
Description
=============
	This program uses inheritance via parent class to calculate properties
	of shapes.

=============
Compiler used
=============
	g++

How to use: If the required files are located within the same folder,
	    then type the following segment without quotations:
		'g++ *.cpp'

	    Otherwise compile each .cpp file individually. 
		ex:
		    'g++ CHpyramid.cpp CHtriangle.cpp CHsquare.cpp 
			 CHpolygon.cpp CHshape.cpp CHcircle.cpp
			 CHcylinder.cpp CervantesMain.cpp  '

=============
How to run
=============
	Once the necessary files are compiled, run the program 
	by typing(without quotes):
		'./a.out'

	Alternatively, because the program has already been compiled, run the
	program by typing(without quotes):
		'./CervantesShapesMain.out'
